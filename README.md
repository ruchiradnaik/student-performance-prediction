# Student Performance Prediction

This project uses machine learning to predict student performance based on various features.

## Setup Instructions
1. Clone the repository.
2. Navigate to the project directory.
3. Install the dependencies using `pip install -r requirements.txt`.
4. Run the model training script using `python model.py`.
5. Start the Flask application using `python app.py`.
6. Send a POST request to `/predict` with the required features to get predictions.

## Features
- Predict student performance based on input features.
- Built with Flask and scikit-learn.
